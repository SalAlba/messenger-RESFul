# messenger-RESFul v. 0.1.0
A simple demo of messenger  in maven  using REST communication (JAX-RS) and Jersey

## Details ##

* DataBase is mock;
* Focusing on REST communication;
* using Jersey lib;
* Postman as REST-client.
